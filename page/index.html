ECScript(ES5)

数据类型

js中的数据类型有基础类型和引用类型， 基础类型有 null, undifined, string, number, boolean, symbol

引用类型有 object

null： 表示人为设置的一个空对象，在内存中的表现是栈中的变量没有指向堆中的内存指针
undifined: 自然状态下的值，比如变量声明未赋值，或者函数定义了函数的形参但是没有实参，访问不存在的变量或者属性

在内存中 基本类型是在栈内存中存储的值，对于引用类型在栈内存中存储的是指向堆内存的指针

平时一般使用typeof 去判断基础类型的数据类型
用 instanceof去判断引用类型的数据类型

instanceof : 判断对象的原型链中是不是能找到类型的 prototype。
