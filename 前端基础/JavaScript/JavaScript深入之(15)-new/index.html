JavaScript深入之new操作符

发表于 2021-08-23 阅读次数：

new 操作符

构造函数实例化通过 new 操作符初始化一个构造函数实例对象，在这个新的对象中为什么可以访问构造函数内的函数和变量？同时实例化之后的对象中的this为什么指向构造函数？

自定义new函数

由此可见 new操作符的实质是

  1. 传建一个空对象
  2. 将空对象中原型对象proto 指向构造函数的原型(prototype)
  3. 使用call 或者 apply 绑定函数内的this指向
  4. 执行构造函数， 如果构造函数的返回值是对象就返回改值，如果不是就返回创建的空对象
