先搞定清楚 prototype 和 proto之间的区别

区别

a, 拥有属性的对象不同

proto: 在js中， 万物皆为对象，对象具有proto属性，可称为隐式原型； 一个对象的隐式原型指向该对象的构造函数的原型（js的对象）

prototype: 函数这个特殊对象所特有的属性（原型属性）（函数）

b, 指向不同

proto 指向它的构造函数的原型对象（prototype ）

该原型对象也有一个自己的隐式原型( proto ) 和 constructor;
constructor: 指向构造函数
proto: 指向它的构造函数的原型对象
